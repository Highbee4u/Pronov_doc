Inventory Register

  1. Basic Data: This data is compulsory and it’s the first step needed to be filled. All the details on this page are marked compulsory and must be filled before proceeding to the next phase. Below is the screenshot of the basic data session of the register inventory page.

  2. More Info: This is the second stage of register inventory page. This page contains additional information on the inventory and its optional ( filling it is not compulsory).
