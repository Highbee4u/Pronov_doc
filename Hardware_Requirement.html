Hardware Requirement


Any system to be used must preferably have the following configuration:


  • A colour monitor with a minimum resolution of 1280 X 800 pixels, preferably an LCD screen
  • A 3GHz processor and above
  • A 3GB memory (RAM) or above.
  • A minimum of 256GB hard disk space.
  • A PS/2 port keyboard or a compatible
  • A Microsoft mouse or any compatible
